# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-XX

### Added
- Initial release of QR Certificate Validator
- PDF processing with QR code extraction
- Multi-algorithm QR detection (OpenCV, pyzbar)
- Online certificate validation via web scraping
- Excel report generation with professional formatting
- Security analysis for PDF files
- Resource monitoring and performance tracking
- Cloud cost analysis for deployment
- Comprehensive test suite
- Security validations (path traversal, SSRF protection)
- Structured logging system
- CLI interface with progress tracking
- Batch processing capabilities
- Docker support preparation
- Comprehensive documentation

### Security
- Path traversal protection
- SSRF prevention with domain whitelisting
- Input validation and sanitization
- PDF malware detection
- Structured security event logging

### Performance
- Memory-efficient PDF processing
- Multi-threading support
- Configurable timeout strategies
- Resource cleanup automation
- Performance monitoring with psutil
